DESCRIPTION
-----------
colorTools is an R package designed to support convenient selection
of colors and shapes while working in RStudio.

The package leverages the learnr tutorial system as a platform
to provide an interactive tool that can remain open alongside the script editor
and console.


HOW IT IS USED
--------------
colorTools runs inside the RStudio Tutorial pane via learnr.
This allows the tool to Stay open while you write and run code.


INSTALLATION
------------
colorTools is currently installed directly from GitHub.

1. Install the remotes package (if not already installed):

   install.packages("remotes")

2. Install colorTools from GitHub:

   remotes::install_github("BoazRosenberg/colorTools")

3. Restart R after installation to ensure the tutorial appears in the "Tutorial"" pane.


STATUS
------
colorTools is under active development.
Palette curation, shape options, and interface behavior may change as the tool
evolves.
